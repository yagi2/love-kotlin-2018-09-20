## setColorの実装
<br />
プロパティっぽくするならプロパティにするしかない  
<br />
→ 拡張プロパティで解決  
  
```Kotlin
var TextView.textColor: Int
    get() = currentTextColor
    set(value) = setTextColor(value)
```
<br />
これでここまで書けるようになった
```Kotlin
linearLayout {
    textView {
        text = "Hello,Kotlin DSL!"
        color = Color.RED
    }.lparams {
        setMargins(dip(10))
    }
}
```
