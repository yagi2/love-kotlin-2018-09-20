## 完成
<br />
これで最初の完成形が動いて、レイアウトが表示できるようになった  
```Kotlin
linearLayout {
    textView {
        text = "Hello,Kotlin DSL!"
        textColor = Color.RED
    }.lparams {
        margin = dip(10)
    }
    
    // コード側でレイアウトを組んでいるのでこういうのも書ける
    (1..10).forEach {
        textView {
            text = it.toString()
        }
    }
}
```
