## TextView
<br />
`textView{}`の中で`text = "text"`を書きたい  
<br />
→ レシーバ指定で`textView{}`のラムダの中の`this`を`TextView`にする
  
```Kotlin
fun textView(init: TextView.() -> Unit): TextView =
    TextView(context).apply(init)
```
<br />
これでここまで書けるようになった  
```Kotlin
linearLayout {
    textView {
        text = "Hello,Kotlin DSL!"
        setColor(Color.RED)
    }
} 
```
