## LayoutParamsの適用
<br />
`LayoutParams`に`width`と`height`を渡せるようにする（0のため）

```Kotlin
fun TextView.lparams(
    width: Int,
    height: Int,
    init: LinearLayout.LayoutParams.() -> Unit
): TextView {
    val lp = LinearLayout.LayoutParams(width, height)
    layoutParams = lp // this.layoutParams
    return this
}
```
